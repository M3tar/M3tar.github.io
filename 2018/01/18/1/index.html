4.RRRSA

首先还是要学习一下RSA算法是什么，下面贴一下学习的地方。

http://bobao.360.cn/learning/detail/3058.html

在这里主要思路就是先通过n计算出p,q，通过p,q再计算出d，之后就可以求出m。
就在我上面贴的网址里面有可以求出p,q的值的网址。

解出来p,q的值。

写一个py脚本跑一下d的值。

def egcd(a, b):
    if a == 0:
        return (b, 0, 1)
    else:
        g, y, x = egcd(b % a, a)
        return (g, x - (b // a) * y, y)
def modinv(a, m):
    g, x, y = egcd(a, m)
    if g != 1:
        raise Exception('modular inverse does not exist')
    else:
        return x % m
